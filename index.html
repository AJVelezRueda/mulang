Mulang

A universal, multi-language, multi-paradigm code analyzer

Mulang is three different - but highly related - things:

  • an intermediate language, known as the Mulang AST;
  • a command line tool for analysing the Mulang AST and some popular languages by transforming to it. This tool is distributed as both a linux-amd64 binary and a JavaScript package. See downloads section.
  • a Haskell composable combinators library for analysing the Mulang AST;
